# ML Engineering Projects

A collection of end-to-end ML projects focusing on reproducibility,
experiment tracking, and production-ready pipelines.

## Projects

- 🚢 [Titanic Survivors](./titanic-survivors)
  - Binary classification (tabular data)
  - sklearn + Hydra + MLflow
  - Threshold optimization and structured error analysis

- 🏭 [Defect Quality Control](./defect-qc)
  - Binary image classification (industrial defect detection)
  - PyTorch + Hydra + MLflow
  - CNN baseline, class imbalance handling, confusion matrix analysis

- 📄 [Financial RAG Assistant](./finantial-rag-assistant/)
  - Retrieval-Augmented Generation (RAG) system for querying PDF documents
  - OpenAI API + FAISS + Streamlit + Docker
  - PDF ingestion, semantic search with embeddings, and interactive question-answering interface
